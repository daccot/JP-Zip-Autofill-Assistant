Privacy Policy for JP Zip Autofill Assistant
Last Updated: 2026-04

Overview
JP Zip Autofill Assistant is a Chrome extension that helps users automatically fill Japanese address forms based on postal codes.

We respect user privacy and do not collect, store, or transmit personal data.

Data Collection
This extension does not collect any personally identifiable information (PII).

The extension temporarily processes the following information locally in the browser:
- Postal codes entered by the user
- Address form field values (for autofill purposes)

This data is not stored externally or transmitted to any servers except when querying external APIs for address lookup.

External API Usage
To provide address lookup functionality, the extension sends postal codes or address queries to publicly available APIs:
- https://zipcloud.ibsnet.co.jp
- https://postcode.teraren.com

These requests are necessary for the core functionality of the extension.

Data Storage
The extension uses Chrome storage (chrome.storage.sync) to store:
- User settings
- Templates
- Custom dictionary entries
- Search history
- Favorites

All stored data remains within the user's browser and is not shared externally.

Permissions Usage
- storage: to save user settings and preferences
- tabs: to interact with the active tab when applying templates
- scripting: to inject scripts for autofill functionality
- host permissions: to access web pages and external APIs for address lookup

Data Sharing
No user data is sold, shared, or transferred to third parties.

User Control
Users can:
- Enable or disable autofill
- Manage templates and dictionary entries
- Clear search history and stored data at any time
